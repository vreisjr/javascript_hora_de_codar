2 - variáveis e tipo de dados

  • Variáveis são formas de como salvamos valores na memória do computador

  • Sintaxe das variáveis: TIPO(var, let ou const) NOME = VALOR;

  • Strings: são textos, usamos '' ou "" para definir

  • No Javascript existem três formas de criar uma variável: var, let e const

    1. var: Forma mais antiga de declarar uma variável

    2. let: Forma mais modernar de declarar uma variável

      3. (Tanto var, quanto let, podem ser alteradas o valor)

    3. const: vem de constante, ou seja, não muda o valor

  • Exitem também os tipos de dados:

    1. Números: Em Javascript, não há diferença entre números inteiros, e números com casas decimais

      2. Exemplo:

      let idade = 25;

      let altura = 1.73;


    2. Texto: São as strings, podem ser formadas tanto por letras, quanto por números, desde que estejam entre '' ou ""

      3. Exemplo:

      let nome = 'Beto';

      let nascimento = '1996';

    3. Booleanos: São apenas true ou false (Mais usados em compação de variáveis e em condicionais)


  • Avaliar o tipo de variável (typeof)

    • Com o comando typeof, é possivel saber qual o tipo da variável
    • Exemplo:

      • let inscrito = 'Ricardo';

      let inscricao = 3447821


      console.log(typeof inscrito);

      console.log(typeof inscricao);


  • Operações Básicas:

    • Concatenar Textos = Unir textos

    • Exemplo:

      • let nome = 'Silvio';

      let sobreNome = 'Santos';

      let nomeCompleto = nome + ' ' + sobreNome;

      ( ' ' é o espaço, para o nome e o sobrenome não ficarem juntos)


    • Operações Aritméticas:

    • Soma +
    • Subtração -
    • Divisão /
    • Multiplicação *

    • Comparações:

    • Maior que >
    • Menor que <
    • Maior ou igual >=
    • Menor ou igual <=
    • Igual ==
    • Identico === (Confere se o valor é igual, e o tipo também)
